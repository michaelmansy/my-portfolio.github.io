Yelp Camp is considered to be my biggest and longest project. It is alos my first project using mongodb I also used some new packages such as passport and mongoose and passport local for authentication. I also used a package called flash which sends flash messages to the user whenever there is any error. This app bascialy lets the users sign up and sees a list of campgrounds posted by any one else. The user also has the capability to add a new campground (once they are logged in). A possibility of leaving a comment is also there. the user has also the possibility to edit or delete his own campground. However, it is not completely finished yet as I am still planning to add some features like a google map and style all the other pages (currently the landing page only is completely done). Once everything is done, Ill add a video walkthrough at the bottom of this page which explains everything in the app.
