Budget App

view project view code

Budget App is condiered to be the biggest front end application I have worked on where I have been able to focus on Javascript more in depth as I worked with some javascript concepts such as: call back functions, IFEE and closures. Concerning the functionality of this app, it basically allows the user to keep track of their incomes and expenses by adding them to the app which will therefore calculate the total of incomes and the total of expenses in addition to the percentage of expenses based on the income. The user also can delete any of the expenses or incomes wrongly entered.
